2.3 Extração de características locais


A atividade de extração de características é responsável por descrever localmente uma imagem em termos das características existentes identificadas (no nosso caso em termos de sua forma).
Primeiro, a imagem é preprocessada e então suas características extraídas e organizadas de forma vetorial.

A representação vetorial da imagem permite a aplicação de métricas para análise de dissimilaridade como a distância Euclideana e a distância de Cossenos.

Para extração das características usamos um descritor de forma baseado nos Polinomios de Zernike.

Descritores de imagem são um conjuntos de técnicas que descrevem imagens em termos de suas características como forma, cor, textura, movimento entre outros. Descritores globais procuram identificar características gerais de uma imagem, enquanto descritores locais procuram características em partes específicas de uma imagem.

Clustering validation

Silhouette plot

Recall that the silhouette coefficient (Si) measures how similar an object i is to the the other objects in its own cluster versus those in the neighbor cluster. Si values range from 1 to - 1:

A value of Si close to 1 indicates that the object is well clustered. In the other words, the object i is similar to the other objects in its group.
A value of Si close to -1 indicates that the object is poorly clustered, and that assignment to some other cluster would probably improve the overall results.
It’s possible to draw silhouette coefficients of observations using the function fviz_silhouette() [factoextra package], which will also print a summary of the silhouette analysis output. To avoid this, you can use the option print.summary = FALSE.

Validation statistics

cluster.number: number of clusters
cluster.size: vector containing the number of points in each cluster
average.distance, median.distance: vector containing the cluster-wise within average/median distances
average.between: average distance between clusters. We want it to be as large as possible
average.within: average distance within clusters. We want it to be as small as possible
clus.avg.silwidths: vector of cluster average silhouette widths. Recall that, the silhouette width is also an estimate of the average distance between clusters. Its value is comprised between 1 and -1 with a value of 1 indicating a very good cluster.

O coeficiente de silhueta (Si) mede a similaridade de um objeto i aos outros objetos em seu próprio cluster em relação aos do cluster vizinho. Valores de Si variam de 1 a - 1:
Um valor de Si próximo a 1 indica que o objeto está bem agrupado.
Um valor de Si próximo a -1 indica que o objeto está mal agrupado e que a designação para algum outro cluster provavelmente melhoraria os resultados.

Fowlkes-Mallows: 0.661
O índice Fowlkes-Mallows (sklearn.metrics.fowlkes_mallows_score) pode ser usado quando as atribuições da classe de verdade do solo das amostras são conhecidas. O escore de Fowlkes-Mallows FMI é definido como a média geométrica da precisão pareada e recordação: FMI = TP (TP + FP) (TP + FN)
Designações de rótulos aleatórias (uniformes) têm uma pontuação FMI próxima a 0.0 para qualquer valor de n_clusters e n_samples (o que não é o caso para Informações mútuas brutas ou a medida V, por exemplo). Limite superior em 1: Valores próximos de zero indicam duas atribuições de rótulo que são amplamente independentes, enquanto valores próximos a um indicam concordância significativa. Além disso, valores de exatamente 0 indicam atribuições de rótulo puramente independentes e um FMI de exatamente 1 indica que as duas atribuições de rótulo são iguais (com ou sem permutação). Nenhuma suposição é feita na estrutura do cluster: pode ser usada para comparar algoritmos de clustering, como k-means, que assume formas de blobs isotrópicos com resultados de algoritmos de cluster espectral que podem encontrar cluster com formas “dobradas”.
